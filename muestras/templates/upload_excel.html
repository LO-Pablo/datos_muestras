Subir archivo Excel

Utilizando la siguiente plantilla, si no se usa dicha plantilla, no se pueden subir las muestras (no usar el excel de errores). En la plantilla están marcadas en rojo las siguientes columnas obligatorias:
  • Id del individuo del cual proviene la muestra
  • Nombre de la muestra asignado por el laboratorio
  • Nombres de las distintas posiciones (congelador, estante...) que sirven para situar a la muestra en el congelador
Si estas columnas obligatorias no se rellenan, la muestra no se subirá a la base de datos. La columna "Estudio" se debe de rellenar con el ID interno del estudio (ID en la tabla de la sección de estudios).
